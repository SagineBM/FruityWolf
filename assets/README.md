# FruityWolf - Assets

This directory contains application assets:

- `icon.png` - Application icon (256x256)
- `icon.ico` - Windows icon file
- `fonts/` - Custom fonts (e.g., Inter)
- `images/` - UI images and illustrations

## Generating Icons

Run the build script to generate placeholder icons:

```bash
python build.py --assets
```

## Recommended Fonts

For the best Spotify-like experience, download Inter font:
- https://fonts.google.com/specimen/Inter

Place the `.ttf` files in the `fonts/` directory.
