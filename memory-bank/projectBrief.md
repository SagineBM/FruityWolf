# Project Brief: FruityWolf

## Core Philosophy
**"Spotify for Producers"** — A beautiful, modern library manager and player for FL Studio project folders.
FruityWolf aims to replace the basic file explorer interaction with a rich, media-centric interface that treats FL Studio projects (`.flp`) and their renders as "tracks" in a library.

## Key Features
- **Modern UI**: Dark theme, glassmorphism, smooth animations (PySide6/QML).
- **Intelligent Indexing**: Auto-scans FL Studio folders, detecting projects and associated audio renders.
- **Instant Playback**: In-app audio player with waveform visualization.
- **Smart Metadata**: BPM & Key detection, tagging system, mood/genre organization.
- **Non-Destructive**: Reads files without modifying project structure.

## User Workflow
1.  **Immerse**: User points FruityWolf to their music production folder.
2.  **Discover**: App indexes projects, creating a searchable library of "tracks" (renders).
3.  **Organize**: User tags ideas ("Sketch", "Finished", "Need Mix"), filters by key/bpm.
4.  **Act**: One-click to open the actual `.flp` in FL Studio or drag-and-drop stems.

## Success Metrics
- **Performance**: Zero-lag scrolling through thousands of projects.
- **Accuracy**: Correctly pairing `.flp` files with their latest audio renders.
- **Aesthetics**: "Wow" factor in UI design, matching modern creative tools.
